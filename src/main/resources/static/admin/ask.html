为什么现在不能提问，或者页面提示知识库还没准备好

通常是因为知识库还没有可用内容，或者同步运行还没完成。先回“知识库管理”页面确认资料是否已经成功入库，再回来提问；如果资料都还没处理完，这里通常也不会稳定出结果。

去知识库管理
为什么回答没有引用来源

先看这次问题是否真的命中了知识库内容。如果答案区有内容但引用为空，优先怀疑检索命中不稳或资料本身还没整理好；这时回“知识库管理”确认对应资料是否存在、标题是否可搜到，会比盲目重问更有效。

去已入库内容
为什么答案不准确或者不完整

先换成更具体的问题，把产品名、流程名、文档名带上；如果还是不准，说明知识库里可能缺资料、旧资料未更新，或者导入后的内容质量不够好。这类问题优先回知识库管理页检查资料，不是先改提问技巧。

回知识库管理
提问时报错了，应该先看哪里

先看页面顶部提示信息，再判断是“知识库没准备好”还是“服务不可用”。如果是内容相关问题，回知识库管理页看最近同步运行；如果是模型、解析或后台配置问题，再去管理员设置，不要一上来就在多个页面来回切。

为什么刚更新资料，答案还是旧的

资料改动后通常需要重新同步或重新进入可问答状态，结果不会立刻变化。先去知识库管理确认最近同步运行是否完成，再回来重新提问。
