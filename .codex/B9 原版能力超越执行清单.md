- 本清单只覆盖 B9，不回退 B5-B8 已收口主链

**设计约束**：

- CLI 默认走独立模式，可选远程模式；不得退化为仅 HTTP 壳
- stdio MCP 默认走独立模式，可选 Bridge 模式；不得要求预启动 HTTP 服务

- 后台治理面必须复用既有 service 层，不新增重复业务逻辑
- B9 所有完成勾选必须有代码、测试或文档依据
